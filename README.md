# Blog 

### 📒 개요
스마일게이트 윈터 데브 캠프 1인 프로젝트 blog 입니다. 기능을 최대한 많이 추가하는 것을 목표로 하고 있으며 MSA를 공부하여 처음으로 MSA 아키텍처로 설계하여 개발하는 것을 목표로 합니다. 개인 기술 블로그로 사용할 계획입니다.
### 📗 기간
2022.12.05. ~
### 📙 Requirements
- 글 쓰기/ 수정
- 글 목록/삭제
- 댓글
- 관리자 도구
- 좋아요
- RSS
